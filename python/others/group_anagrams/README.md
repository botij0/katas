# Enunciado

Given an array of strings, group the anagrams together.

```python
groupAnagrams(["eat", "tea", "tan", "ate", "nat", "bat"])
sol:[["eat","tea","ate"],["tan","nat"],["bat"]]

groupAnagrams(["vote", "please"])
sol: [["vote"],["please"]]

groupAnagrams(["debitcard", "badcredit"])
sol: [["debitcard", "badcredit"]] 
```
